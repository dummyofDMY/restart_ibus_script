# ibus 重启脚本
## 作用
用于快速重启ibus，解决键盘卡死的问题。这个现象似乎会反复出现，于是写了个脚本。
## 使用方法
### 只使用脚本
直接执行仓库里的.sh脚本就行，在仓库目录开启终端执行：
```bash
bash restart_ibus.sh
```
但核心矛盾是， **你键盘坏了要怎么把指令输入到终端里？**  
~~可能可以用鼠标复制粘贴？~~
### 桌面快捷方式
为了不用键盘快速执行，将仓库里的.desktop放到桌面上，用文本编辑器打开这个文件修改`Exec`、`Path`、`Icon`三个字段的路径，让它们正确指向对应文件，然后右键.desktop文件设为“允许执行”，以后就能通过`super+D`退到桌面，双击这个文件直接执行。
